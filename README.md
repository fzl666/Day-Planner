# Day-Planner

A planner for working hours.

## The planner

[Quick acess](https://fzl666.github.io/Day-Planner/)

## Features

Plan and save your schedule between 9 a.m to 5 p.m.

Renew at 6 p.m. so you can plan for the next day beforehand.
